# ude-proyecto
ude-proyecto

## Juego
Es necesario tener intalado nodejs.
Luego de clonar el repo, abrira la carpeta "proyecto_juego" en una consola e instalar dependencias.
```
npm install
```

Para correrlo
```
npm run dev
```

Va acrear un web server que indicara la url
```
https://localhost:8080/
```
